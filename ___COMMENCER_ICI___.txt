🔧 OUTILS AUTOMATIQUES
════════════════════════════════════════════════════════════════════

Pour vous faciliter la vie, utilisez ces scripts PowerShell :

┌────────────────────────────────────────────────────────────────┐
│ ✅ Vérifier votre configuration réseau                         │
└────────────────────────────────────────────────────────────────┘

   Dans PowerShell, tapez :
   
   .\VERIFIER_CONFIG_RESEAU.ps1
   
   Ce script vous dit :
   ✓ Votre adresse IP
   ✓ Si le pare-feu est configuré
   ✓ Si l'application tourne
   ✓ La configuration CORS


┌────────────────────────────────────────────────────────────────┐
│ 🔥 Créer les règles de pare-feu automatiquement                │
└────────────────────────────────────────────────────────────────┘

   Dans PowerShell EN ADMINISTRATEUR, tapez :
   
   .\CREER_REGLES_PAREFEU.ps1
   
   Ce script crée automatiquement :
   ✓ Règle pour le Backend (Port 5000)
   ✓ Règle pour le Frontend (Port 5173)
